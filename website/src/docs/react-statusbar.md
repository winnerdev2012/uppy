The `<StatusBar />` component wraps the [StatusBar][] plugin.

```js
import StatusBar from 'uppy/lib/react/StatusBar';
```

## Props

The `<StatusBar />` component supports all [StatusBar][] options as props.

```js
<StatusBar
  hideUploadButton
  hideAfterFinish={false}
  showProgressDetails
/>
```
